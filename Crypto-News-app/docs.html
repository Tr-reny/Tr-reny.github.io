Introduction

Crypto News App is a professional open source news app built using Java and XML. It offers a user-friendly interface with lots of options for customization. Get all the features of a news app including live updates, latest news, and customizable categories.

This project can be used as a guide for learning Java and XML while building a news app.

Features#

  • Made with Java and XML
  • User-friendly Interface
  • Live News Updates
  • Customizable Categories
  • Latest News
  • News Search
  • Saved Articles
  • News Sharing
  • And Much More…

Technologies#

  • Java
  • XML

Contributing#
